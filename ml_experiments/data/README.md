# ML Experiments — Data Directory

This directory stores subsets of the Spider dataset used for training and evaluation.

## Expected Files

- `spider_train.json` — Training split of the Spider dataset
- `spider_dev.json` — Development/validation split
- `spider_tables.json` — Database schema definitions

## Obtaining the Dataset

The Spider dataset can be downloaded from:
https://yale-lily.github.io/spider

> **Note:** The dataset files are excluded from version control via `.gitignore`.
> Download and place them here manually.
